# Лабораторна робота №1
Дисципліна: "Кросплатформне програмування"
Виконала: Одинець Вероніка

## Опис проєкту
Проєкт присвячений вивченню Java Reflection API.
Реалізовано наступні завдання:
* `ClassAnalyzer.java` — консольний аналізатор структури класу.
* `ClassAnalyzerGUI.java` — графічний інтерфейс для аналізу класів.
* `ObjectInspector.java` — динамічне дослідження стану об'єктів та виклик методів.
* `MethodInvoker.java` — виклик методів за рядковим іменем.
* `ArrayManipulator.java` — динамічне керування масивами та матрицями.
* `ProxyDemo.java` — використання динамічних проксі для профілювання та трасування.
